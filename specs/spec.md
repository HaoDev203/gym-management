# 健身房管理系统 - 产品需求规格说明书

## 1. 项目概述

### 1.1 项目背景
本系统是一套基于 Web 的企业级健身房管理系统，旨在帮助健身房实现会员管理、课程预约、教练管理、器材管理等核心业务的数字化管理。

### 1.2 目标用户
- **会员**：健身消费者，可预约课程、查看个人信息
- **管理员**：健身房工作人员，负责系统运营管理

---

## 2. 用户角色与权限

### 2.1 会员（Member）
- **注册方式**：用户自行在线注册，注册完成即可使用系统
- **有效期**：永久有效
- **等级体系**：无等级区分，所有会员权限相同

**会员权限：**
- 查看可预约课程列表
- 预约课程（团课/私教课）
- 取消预约（开课前 2 小时以上）
- 查看个人预约记录
- 查看和修改个人基本信息
- 查看课程价格

### 2.2 管理员（Admin）
- **分级管理**：
  - **超级管理员**：可管理管理员账号（创建、修改、删除），拥有所有权限
  - **普通管理员**：可管理会员、课程、器材、员工，不可管理管理员账号

**管理员权限：**
- 会员管理（查看、编辑、删除会员信息）
- 员工管理（教练信息管理）
- 课程管理（创建、编辑、删除课程，设置容量、时间、价格）
- 器材管理（器材清单维护）
- 订单管理（查看预约记录、线下收费记录）
- 数据统计报表查看

---

## 3. 核心业务模块

### 3.1 会员管理

**会员信息字段：**
- 姓名
- 手机号
- 身份证号
- 性别
- 生日
- 紧急联系人姓名
- 紧急联系人电话

**功能要求：**
- 会员可自助注册账号
- 会员可查看和修改个人基本信息
- 管理员可查看、编辑、删除会员信息
- 会员账号永久有效

### 3.2 员工管理

**员工定义：** 健身教练

**员工信息字段：**
- 姓名
- 联系方式
- 擅长课程（可多选）
- 可排班时间

**功能要求：**
- 管理员可创建、编辑、删除教练信息
- 教练信息用于课程排课和会员选择

### 3.3 器材管理

**器材信息字段：**
- 器材名称
- 数量
- 位置（如：一楼有氧区、二楼力量区）

**功能要求：**
- 管理员可添加、编辑、删除器材信息
- 器材信息仅用于展示，不追踪使用状态和维护记录

### 3.4 课程管理

**课程类型：**
- **团课**：一个教练对多个会员（如瑜伽、动感单车、搏击操等）
- **私教课**：一个教练对一个会员

**课程信息字段：**
- 课程名称
- 课程类型（团课/私教课）
- 教练
- 上课时间（开始时间、结束时间）
- 课程容量（团课必填，私教课固定为 1）
- 课程价格（团课价格低于私教课）
- 可预约截止时间（开课前 30 分钟）

**功能要求：**
- 管理员可创建、编辑、删除课程
- 会员可查看可预约课程列表
- 会员可预约课程（需符合预约限制规则）
- 会员可取消预约（开课前 2 小时以上）
- 课程满员后支持候补排队
- 候补排队成功时，系统自动通知候补会员

### 3.5 预约与订单

**预约流程：**
1. 会员选择目标课程
2. 系统校验预约资格（时间冲突、预约数量限制等）
3. 预约成功，生成订单
4. 会员上课签到
5. 线下支付费用

**订单信息字段：**
- 订单编号
- 会员 ID
- 课程 ID
- 预约时间
- 订单状态（已预约/已取消/已完成/已爽约）
- 应付金额
- 实付金额（线下填写）
- 支付状态（未支付/已支付）

**预约限制规则：**
- **时间冲突检测**：同一时间段不能预约多门课程
- **预约数量限制**：每人每天最多预约 3 节课
- **提前预约**：最多可提前 7 天预约
- **临时预约**：开课前 30 分钟截止预约

**取消规则：**
- 开课前 2 小时以上可取消
- 开课前 2 小时内不可取消

**签到规则：**
- 会员上课需签到
- 爽约（未签到）记录到会员账号
- 累计爽约 3 次后，限制预约权限 7 天
- 爽约记录永久保存，管理员可手动清除

**收费方式：**
- 线上仅展示价格和预约，不涉及支付
- 所有费用线下支付（前台收费）
- 管理员可在订单中记录实付金额

### 3.6 通知提醒

**通知方式：**
- 站内信
- 短信

**通知场景：**
- 预约成功通知
- 取消预约通知
- 课程开始前提醒（提前 1 小时）
- 课程即将开始提醒（提前 5 分钟）
- 候补排队成功通知
- 教练/管理员取消课程通知

## 4. 数据统计与报表

管理员后台需提供以下统计功能：

### 4.1 课程报名统计
- 每日/每周/每月课程报名数量
- 各类型课程（团课/私教课）报名占比
- 热门课程排行

### 4.2 会员活跃度统计
- 每日/每周/每月活跃会员数
- 会员预约频次分布
- 会员流失率

### 4.3 教练上课统计
- 教练上课课时统计
- 教练课程满员率
- 教练受欢迎程度排行

### 4.4 收入统计
- 每日/每周/每月线下收费汇总
- 各课程类型收入占比
- 收入趋势分析

---

## 5. 边缘场景处理

| 场景 | 处理规则 |
|------|----------|
| 课程已满 | 会员可选择候补排队，有人取消时自动通知候补会员 |
| 教练临时请假 | 管理员取消课程，系统通知已报名会员 |
| 同一会员同一时间段报名多门课程 | 系统校验时间冲突，禁止重复预约 |
| 会员报名后未签到（爽约） | 记录爽约，累计 3 次后限制预约 7 天 |
| 器材故障导致课程无法进行 | 管理员调课或取消课程，通知已报名会员 |
| 会员取消课程（开课前 2 小时内） | 系统禁止取消 |

---

## 6. 非功能性需求

### 6.1 性能要求
- 课程列表加载时间 < 2 秒
- 预约操作响应时间 < 1 秒

### 6.2 可用性要求
- 系统支持 7×24 小时运行
- 支持并发预约（多人同时预约同一课程）

### 6.3 数据一致性要求
- 预约冲突检测必须准确，禁止超卖
- 订单状态变更需记录操作日志
